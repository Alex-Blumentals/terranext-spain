# PDF Export Instructions

## How to Download the Presentation as PDF

### Method 1: Browser Print (Recommended)

1. Open the presentation in your browser: `https://presentacion.terranext.co`
2. Add `?print-pdf` to the URL: `https://presentacion.terranext.co?print-pdf`
3. Wait for all slides to load completely
4. Press `Ctrl+P` (Windows/Linux) or `Cmd+P` (Mac) to open Print dialog
5. Select "Save as PDF" as the printer
6. Configure settings:
   - **Layout:** Portrait
   - **Margins:** None
   - **Background graphics:** Enabled (Important!)
7. Click "Save" and choose your destination

### Method 2: Direct Print from Reveal.js

1. Open the presentation: `https://presentacion.terranext.co`
2. Add `?print-pdf` to the URL
3. The page will automatically format for PDF
4. Use browser's print function (Ctrl+P / Cmd+P)
5. Save as PDF

## Tips for Best PDF Quality

- **Wait for fonts to load:** Give the page 5-10 seconds after loading before printing
- **Enable background graphics:** This ensures all colors and gradients appear in PDF
- **Check all slides:** Scroll through to ensure nothing is cut off
- **File size:** The PDF will be approximately 2-5 MB depending on content

## Troubleshooting

**Problem:** Colors don't appear in PDF
**Solution:** Ensure "Background graphics" is enabled in print settings

**Problem:** Text is cut off
**Solution:** Add `?print-pdf` to URL before printing

**Problem:** Logos don't appear
**Solution:** Wait 10 seconds for assets to load, then try again
